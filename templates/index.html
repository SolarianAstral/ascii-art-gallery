What does Ascii mean?

ASCII (which stands for "American Standard Code for Information Interchange") is a character encoding standard for electronic communication. ASCII codes represent text in computers, telecommunications equipment, and other devices. ASCII has just 128 code points, of which only 95 are printable characters, which severely limit its scope. The set of available punctuation had significant impact on the syntax of computer languages and text markup. ASCII hugely influenced the design of character sets used by modern computers, including Unicode which has over a million code points, but the first 128 of these are the same as ASCII.)

ASCII characters can be used to create ASCII Art, a technique to create images and various figures using Ascii characters. On this website you will find a gallery filled with Ascii images, as well as a converter to turn any image you want into Ascii Art, hope you find it cool :D !!
